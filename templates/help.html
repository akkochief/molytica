Molytica Chemical Analyzer kullanım kılavuzu ve sık sorulan sorular

Molytica Chemical Analyzer, kimyasal veri analizi yapmanızı, tahminler oluşturmanızı ve görselleştirmelerle raporlar hazırlamanızı sağlayan güçlü bir araçtır. Bu rehber, uygulamanın tüm özelliklerini adım adım açıklamaktadır.

Sistem Gereksinimleri

Uygulamayı sorunsuz çalıştırmak için aşağıdaki gereksinimlerin karşılanması önerilir:

  • Tarayıcı: Google Chrome, Mozilla Firefox, Safari veya Microsoft Edge (en son sürümler)
  • İşletim Sistemi: Windows 10+, macOS 10.15+ veya Linux (Ubuntu 20.04+)
  • Ek Araçlar: Python 3.8+ (veri analizi için) ve Node.js 14+ (arka uç işlemleri için)
  • İnternet Bağlantısı: Uygulama çevrimiçi çalıştığı için stabil internet bağlantısı gereklidir

Desteklenen Dosya Formatları

  • CSV — Tabular veriler için standart format. Her kolon başlığı düzgün etiketlenmelidir.
  • Excel (.xlsx, .xls) — Excel dosyalarını CSV'ye dönüştürerek yükleyebilirsiniz.
  • Manuel Veri Girişi — Küçük veri setleri için doğrudan veri girişi yapabilirsiniz.

Nasıl Kullanılır?

1. Veri Yükleme

  • Manuel Veri Girişi: "Manuel Veri Girişi" butonuna tıklayarak verilerinizi elle girin. Küçük veri setleri için idealdir.
  • CSV Dosyası Yükle: "CSV Dosyası Yükle" butonu ile bilgisayarınızdan bir CSV dosyası seçin.
  • Excel'den CSV'ye Dönüştür: Excel dosyasını seçin, sistem otomatik olarak CSV'ye dönüştürecektir.

2. Veri Analizi

  • Katalizör, Verim Tahmini: Kimyasal reaksiyonlar için katalizör ve verim tahmini yapar.
  • Tahmin Yap: Yüklediğiniz veriler üzerinden makine öğrenmesi ile tahminler üretir.
  • Görselleştirme: Verilerinizi grafikler ve diyagramlar ile görsel olarak analiz edin.
  • Model Karşılaştırması: Farklı AI modellerini karşılaştırın ve performanslarını değerlendirin.

3. Raporlama

  • Analiz sonuçlarınızı otomatik olarak rapor haline getirin.
  • Raporlar, veri analizi, tahmin sonuçları ve görselleştirmeleri içerir.
  • Raporları CSV veya PDF formatında dışa aktarabilirsiniz.

Veri Seti Gereksinimleri

  • Zaman Bilgisi: Zaman serisi tahmini yapıyorsanız, veriniz tarih/saat içermelidir.
  • Kategorik Özellikler: Kimyasal tür, bölge gibi kategorik değişkenler bulunmalıdır.
  • Sayısal Özellikler: Sıcaklık, yoğunluk, konsantrasyon gibi sayısal değişkenler gereklidir.
  • Eksik Veriler: Eksik veriler temizlenmeli veya uygun şekilde doldurulmalıdır.
  • Kolon Başlıkları: Tüm kolonlar açık ve anlaşılır şekilde etiketlenmelidir.

İpucu: En iyi sonuçlar için verinizi yüklemeden önce temizleyin ve normalize edin. Eksik değerleri doldurun veya çıkarın.

Sık Sorulan Sorular

  • Hangi veri boyutlarını işleyebilir? — 10.000 satıra kadar veriyi sorunsuz işleyebilir.
  • Sonuçları nasıl dışa aktarabilirim? — Her analiz sonucunda "İndir" butonu ile CSV veya PDF olarak kaydedebilirsiniz.
  • Özel modeller ekleyebilir miyim? — Geliştirici modu ile kendi modellerinizi entegre edebilirsiniz.

Molytica Chemical Analyzer is a powerful tool for chemical data analysis, generating predictions, and creating reports with visualizations. This guide explains all features step by step.

System Requirements

The following requirements are recommended for smooth operation:

  • Browser: Latest versions of Google Chrome, Mozilla Firefox, Safari, or Microsoft Edge
  • Operating System: Windows 10+, macOS 10.15+, or Linux (Ubuntu 20.04+)
  • Additional Tools: Python 3.8+ (for data analysis) and Node.js 14+ (for backend)
  • Internet Connection: Stable internet connection required for online operation

Supported File Formats

  • CSV — Standard format for tabular data. Each column must be properly labeled.
  • Excel (.xlsx, .xls) — Convert Excel files to CSV format for upload.
  • Manual Data Entry — Direct data input for small datasets.

How to Use?

1. Data Upload

  • Manual Data Entry: Click "Manual Data Entry" button to enter data manually.
  • Upload CSV File: Select a CSV file from your computer using the "Upload CSV" button.
  • Convert Excel to CSV: Select an Excel file, and the system will convert it automatically.

2. Data Analysis

  • Catalyst, Yield Prediction: Predicts catalysts and yields for chemical reactions.
  • Make Prediction: Generates predictions using machine learning on uploaded data.
  • Visualization: Analyze data visually with charts and diagrams.
  • Model Comparison: Compare different AI models and evaluate their performance.

3. Reporting

  • Automatically generate reports from analysis results.
  • Reports include data analysis, prediction results, and visualizations.
  • Export reports in CSV or PDF format.

Dataset Requirements

  • Time Information: For time series forecasting, include date/time data.
  • Categorical Features: Include categorical variables like chemical type, region.
  • Numeric Features: Include numeric variables like temperature, concentration.
  • Missing Data: Clean missing data or fill appropriately.
  • Column Headers: All columns must be clearly labeled.

Tip: Clean and normalize your data before upload for best results. Fill or remove missing values.

FAQ

  • What data sizes can be processed? — Up to 10,000 rows without issues.
  • How to export results? — Use the "Download" button to save as CSV or PDF.
  • Can I add custom models? — Yes, with developer mode you can integrate your own models.
